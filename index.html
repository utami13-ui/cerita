Cerita Semester Satu

Pada semester ini, saya mulai beradaptasi dengan lingkungan belajar yang baru dan materi pelajaran yang semakin menantang. Saya belajar banyak hal, terutama yang berkaitan dengan pemrograman dan logika. Walaupun pada awalnya terasa sulit, tetapi saya berusaha untuk memahami materi tersebut sebisa mungkin. Dari semester satu ini, saya belajar pentingnya disiplin, tanggung jawab, dan kerja sama dengan teman teman.


Pada pembelajaran KKPW yang diajar oleh pak Unggah, saya senang dengan cara dia mangajar dan tegas terhadap anak anak yang melanggar aturan. Caranya menjelaskan suatu materi bisa saya pahami dengan cukup baik dan ditambah saya memang menyukai pembelajaran tentang web jadi bisa dibilang ini merupakan mata pelajaran yang paling bisa saya kuasai daripada mata pelajaran produktif lainnya. Di semester satu ini pak Unggah tidak hadir beberapa kali, jadi saya berharap semoga di semester selajutnya pak Unggah bisa hadir lebih sering agar saya bisa belajar lebih banyak lagi tentang web. Selama semester satu ini saya sangat senang karena saya bisa belajar mengenai html, css dan javascript yang sangat menarik bagi saya. Pak Unggah juga memberikan beberapa projek yang dapat melatih kemampuan saya dalam memahami html, css dan javascript.


Pada mata pelajaran lainnya juga saya cukup senang mengikutinya, walaupun ada beberapa guru yang cara mengajarnya kurang saya pahami. Namun, saya tetap berusaha untuk mengikuti pelajaran dengan baik dan memahami materi yang disampaikan. Pada semester ini juga ada beberapa kejadian yang kurang menyenangkan bagi saya, yaitu saya mengalami kecelakaan saat pulang sekolah yang mengakibatkan jari telunjuk tagan kanan saya tulangnya mengalami pergeseran dan sulit untuk mengetik untuk sementara waktu. Selain itu, saya juga aktif mengikuti kegiatan ekstrakurikuler di sekolah, seperti bulutangkis.

Cerita Selama Liburan Semester

Selama liburan, saya memanfaatkan waktu untuk beristirahat, membantu orang tua karena banyak ada hari raya agama hindu seperti purnama, tilem dan tumpek. Pada saat liburan saya juga selalu membersihkan rumah saya di pagi hari saat sempat.


Saya juga berlibur ke beberapa tempat seperti, Bedugul, Kintamani, dan beberapa tempat wisata di Bali bersama keluarga. Ada kejadian yang kurang menyenangkan juga karena saat saya membeli beberapa makanan, rasa makanan tersebut tidak sesuai dengan selera saya. Selama liburan ini saya juga belajar memasak beberapa makanan, saya juga membeli beberapa kembang api untuk tahun baruan bersama keluarga saya, saya sangat senang karena kembang apinya sangat bagus dan tidak ada yang gagal, karena tahun lalunya tangan saya terkena kembang api yang gagal meledak.


Saya cukup puas dengan liburan kali ini karena bisa menghabiskan banyak waktu bersama keluarga dan teman teman saya, serta banyak hal menyenangkan dibandingkan dengan hal yang tidak menyenangkan yang terjadi.
